Términos y condiciones.

Condiciones generales para el uso del sitio y compra de productos Taploe.

1. Definiciones

Para efectos de estos términos:

Taploe se refiere a la marca, sitio web, plataforma, productos físicos, perfiles digitales, tarjetas NFC, servicios digitales y soluciones relacionadas.

Cliente o usuario se refiere a cualquier persona física o moral que navega el sitio, compra productos, crea una cuenta, activa tarjetas, administra perfiles o utiliza los servicios de Taploe.

Sitio web se refiere a taploe.com y cualquier subdominio, página, formulario, landing page o sección relacionada con Taploe.

Plataforma Taploe se refiere al sistema digital desde el cual el cliente puede activar tarjetas, administrar perfiles digitales, actualizar información, cargar logotipos, visualizar métricas, configurar enlaces y utilizar funciones disponibles según el producto o plan contratado.

Tarjeta NFC se refiere a una tarjeta física con tecnología NFC y/o código QR que permite abrir un perfil digital, enlace o recurso configurado dentro de la plataforma Taploe.

Tarjeta NFC estándar se refiere a una tarjeta física, generalmente de PVC u otro material similar, que puede incluir NFC, QR, logotipo y elementos visuales definidos por Taploe.

Tarjeta NFC metálica se refiere a una tarjeta física con acabado metálico o premium que puede incluir NFC, QR, grabado o personalización visual conforme a las opciones disponibles.

Perfil digital se refiere a una página digital administrable desde la plataforma Taploe, donde el cliente puede mostrar y actualizar información profesional, datos de contacto, redes sociales, enlaces, botones, fotografía, logotipo, QR y otros elementos disponibles.

Taploe Pro se refiere a una actualización mensual por usuario, tarjeta o perfil, que puede incluir funciones avanzadas como métricas, visitas, clics, enlaces personalizados, campañas, reportes o herramientas adicionales.

Taploe para equipos se refiere a soluciones empresariales o grupales para administrar múltiples usuarios, perfiles, tarjetas, permisos, marca, reportes y configuraciones desde un entorno centralizado.

Contenido del cliente se refiere a información, logotipos, fotografías, textos, enlaces, datos de contacto, archivos, imágenes, marcas o cualquier material proporcionado, cargado o actualizado por el cliente dentro de la plataforma o durante el proceso de compra.
